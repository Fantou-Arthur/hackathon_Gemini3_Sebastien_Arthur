## Version Française 🇫🇷

### 🚀 Présentation
Ce projet est une expérience WebVR immersive où l'intelligence artificielle Gemini (Google DeepMind) agit à la fois comme **Architecte** du monde et comme **Habitant** interactif. L'utilisateur peut générer des univers entiers par simple commande textuelle et interagir avec un agent doué de conscience spatiale.

### 🌟 Fonctionnalités Clés

#### 🏗️ L'Architecte (Gemini 3.1 Pro)
- **Génération Procédurale** : Créez des scènes 3D complexes (villes, fermes, paysages abstraits).
- **Expansion Dynamique** : Ajoutez des éléments à la scène existante sans réinitialiser l'environnement.
- **Sculpture d'Entités** : L'IA ne crée pas de simples blocs, mais sculpte des PNJs (vaches, robots) à partir de primitives multiples.

#### 🤖 L'Agent Gemini (Gemini 2.5 Flash Lite)
- **Cerveau Autonome** : L'agent pense et se déplace chaque seconde.
- **Perception Spatiale** : Il "voit" les objets autour de lui, connaît leur distance et leur direction.
- **Exploration Proactive** : S'ennuie s'il ne se passe rien et part explorer les environs.
- **Interactions Sociales** : Capable de s'approcher des PNJs et de leur parler.
- **Commandes par Chat** : Obéit à vos ordres de mouvement ("Va à gauche", "Vient me voir") tout en discutant.

#### 🛠️ Gestionnaire de NPCs
- **Contrôle en Temps Réel** : Une interface dédiée pour surveiller chaque entité du monde.
- **Mouvement Groupé** : Tous les habitants sont régis par une boucle de pensée collective IA.
- **Bouton Panic** : Réinitialisation instantanée des positions si les PNJs s'égarent.

### 🛠️ Installation
1. Clonez le dépôt.
2. Créez un fichier `.env` à la racine avec : `GEMINI_API_KEY=votre_clé_ici`.
3. Lancez un serveur local (ex: `python -m http.server` ou `Live Server`).
4. Ouvrez `index.html`.

## English Version 🇺🇸

### 🚀 Overview
This project is an immersive WebVR experience where Gemini AI (Google DeepMind) acts as both the **World Architect** and an **Interactive Inhabitant**. Users can generate entire universes through text prompts and interact with an agent possessed of spatial awareness.

### 🌟 Key Features

#### 🏗️ The Architect (Gemini 3.1 Pro)
- **Procedural Generation**: Create complex 3D scenes (cities, farms, abstract landscapes).
- **Dynamic Expansion**: Add elements to the existing scene without resetting the environment (sky/ground).
- **Entity Sculpting**: The AI doesn't just place blocks; it sculpts NPCs (cows, robots) from multiple primitives.

#### 🤖 The Gemini Agent (Gemini 2.5 Flash Lite)
- **Autonomous Brain**: The agent thinks and moves every second.
- **Spatial Perception**: It "sees" surrounding objects, knows their distance and direction.
- **Proactive Exploration**: Gets bored if nothing happens and sets out to explore the surroundings.
- **Social Interactions**: Capable of approaching NPCs and talking to them.
- **Chat Commands**: Obeys your movement commands ("Go left", "Come see me") while chatting.

#### 🛠️ NPC Manager
- **Real-time Control**: A dedicated interface to monitor every entity in the world.
- **Group Movement**: All inhabitants are governed by an AI collective thought loop.
- **Panic Button**: Instant position reset if NPCs stray too far.

### 🛠️ Setup
1. Clone the repository.
2. Create a `.env` file in the root with: `GEMINI_API_KEY=your_key_here`.
3. Start a local server (e.g., `python -m http.server` or `Live Server`).
4. Open `index.html`.
